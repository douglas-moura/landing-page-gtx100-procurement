# GTX100 Procurement Landing Page

Landing page institucional desenvolvida para apresentar as soluções de Procurement da GTX100.

> Não é comprar mais. É comprar melhor.

---

## 📋 Sobre o projeto

Este projeto consiste em uma landing page responsiva criada para comunicar a proposta de valor da GTX100 Procurement, destacando:

- Estratégia de compras
- Redução de custos
- Gestão de fornecedores
- Inteligência em Procurement
- Áreas de atuação
- Cases e diferenciais
- Geração de leads

---

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- TypeScript
- Tailwind CSS
- PostCSS
- Webpack

## 🚀 Como executar o projeto

### Clone o repositório

```bash
git clone https://github.com/douglas-moura/landing-page-gtx100-procurement.git

cd landing-page-gtx100-procurement
```

### Instale as dependências

```bash
npm install
```

### Ambiente de desenvolvimento

Inicia o servidor local:

```bash
npm run serve
```

---

## 🔨 Scripts disponíveis

### Build para produção

```bash
npm run build
```

### Build para desenvolvimento

```bash
npm run build:dev
```

### Servidor local

```bash
npm run serve
```

### Observa alterações nos arquivos

```bash
npm run watch
```
